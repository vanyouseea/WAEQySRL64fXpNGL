# MJJ扫域名脚本
## 功能
- 自动分离域名, 并判断
- 自动验证域名可以用作干什么
- 自动保存已扫描到的域名
- 接入Tor用于反爬虫
- 忘了
## 不足
- 不会自动停止
- 快过期域名不会保存(可以加, 等更新)
- 可能会抽风
- 凑数
- 强迫症
